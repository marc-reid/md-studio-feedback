# MD Studio feedback

This is the public feedback tracker for [MD Studio](https://mdstudio.dev), a browser-based HTML-to-Markdown converter and Markdown editor.

- [Report a bug](https://github.com/marc-reid/md-studio-feedback/issues/new?template=bug-report.yml)
- [Suggest an improvement](https://github.com/marc-reid/md-studio-feedback/issues/new?template=feature-request.yml)
- [Browse existing reports](https://github.com/marc-reid/md-studio-feedback/issues)

## Privacy

MD Studio processes your document in your browser. It does not send document content to an MD Studio server. If you enable local saving, the document is stored in that browser on that device.

GitHub Issues are different: reports submitted here are public. Do not paste confidential documents, personal information, credentials, or other sensitive material into an issue. Use a small, redacted example when sample content is needed to reproduce a problem.

## What makes a useful report

For bugs, please include:

- what you were trying to do;
- the steps that reproduce the problem;
- what you expected and what happened instead;
- your browser and operating system;
- a small, redacted sample when the problem concerns conversion output.

Before opening a report, please search the existing issues in case somebody has already reported it.

## Security reports

Do not report suspected security vulnerabilities in a public issue. Use [GitHub's private vulnerability reporting](https://github.com/marc-reid/md-studio-feedback/security/advisories/new) instead.
